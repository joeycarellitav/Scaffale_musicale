Configurazione necessaria

Questo sito condiviso ha bisogno di un progetto Firebase (gratuito) per salvare gli utenti e i brani. Apri il file scaffale-musicale.html in un editor di testo, cerca il blocco CONFIGURAZIONE FIREBASE in cima allo script e inserisci i valori del tuo progetto.

Le istruzioni complete passo-passo sono nel file SETUP.md incluso insieme a questo sito.
